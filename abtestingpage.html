A/B Testing

PROBLEM: How does design layout impact the efficiency of a page?

In this project, I explored how alternating the design layout of a page could influence how users efficiently interacted with a page. This is important, because the efficiency of pages can largely affect user experience. The process included the following steps:

  1. Creating 2 pages with different layouts and design details
  2. Determining metrics to measure efficiency and hypothesizing about the two pages
  3. Collecting data from users using both pages randomly
  4. Running statistical tests on the determined metrics
  5. Interpreting the data calculated from the statistical tests

I made 2 websites that sold cacti, with the following different features:

To measure efficiency, I ran statistical tests on two metrics for the two different pages: the time to completion of users adding $150 worth of cacti to their cart, and the users' rate of return to the home page. My hypotheses were:

Time to Completion
Null Hypothesis:

The time to completion (i.e. the time it takes to order at least $150 of cacti) for Interface A is no different from the time to completion of Interface B.

Alternative Hypothesis:

The time to completion for Interface A should be less than the time to completion for Interface B.

Return Rate
Null Hypothesis:

The return rate (i.e. the number of times a user returned to the browsing page from the checkout page) for Interface A is no different from the return rate of Interface B.

Alternative Hypothesis:

The return rate for Interface A should be lower than the return rate for Interface B.

KEY TAKEAWAYS

  • The results of A/B testing are dependent on having sufficient data -- my experiment results could have been more accurate if I had more data sets. Having similar numbers of data sets for Interface A and B is also important, as having more of one interface could skew the data.
  • Since the time to completion of my Interface A (grid layout) is less than that of Interface B (vertical layout), having a layout where users can see items without additionally scrolling is a more efficient design.
  • This experiment demonstrated the importance of good design principles (ex. page layout, affordances, colors, content hierarchy) because they do have a statistically significant impact on how users interact with the page.
